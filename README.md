# 琏雾平台使用说明

## 平台简介

琏雾平台是由中国海洋大学智能物联网团队研发的物联网操作系统，支持在Windows环境下快速开发和运行物联网应用系统，实现系统间互联互通，促进大规模物联网数据汇聚和人工智能发展。

**官方网站**：http://lianwu985.com

------

## 一、安装前注意事项

### 1.1 运行环境冲突

系统包含以下服务，安装前需关闭电脑中已存在的相同服务：

- MySQL
- Nginx
- Rabbitmq
- Redis
- Emqx

**解决方案**：

- 通过任务管理器停止冲突服务
- 或修改现有服务的端口号
- 如需同时使用，可通过启停服务方式切换

### 1.2 虚拟机配置要求

如无法解决冲突，可在虚拟机中安装，配置要求：

- 内存：≥8G
- 磁盘空间：≥50G
- 处理器：≥4核
- 操作系统：Win10及以上版本

------

## 二、使用流程概览

1. 访问琏雾平台网站
2. 注册平台账号
3. 下载系统安装包
4. 申请系统序列号
5. 申请应用序列号
6. 安装并激活琏雾系统
7. 激活物联网应用

> 注：系统使用期限为30天，续期需联系工作人员。

------

## 三、详细操作步骤

### 3.1 网站访问与账号注册

1. 访问 http://lianwu985.com
2. 点击登录→立即注册
3. 填写真实信息（姓名、电话、单位、邮箱）
4. 完成注册并登录

### 3.2 下载安装包

- 登录后点击"下载"，获取LianwuOS-1.0.2.exe（约1.1G）
- 忽略浏览器安全提示，手动通过下载

### 3.3 序列号申请

**系统序列号**：

1. 登录平台→软件激活→序列号申请
2. 选择数量并提交申请
3. 等待审核（1个工作日内）
4. 在"序列号管理"中查看审核结果

**应用序列号**（以安防监控软件为例）：

1. 进入应用商店→选择SecuMS软件
2. 点击申请序列号
3. 等待审核并通过序列号管理查看

### 3.4 系统安装与激活

1. 双击安装包，按提示安装（勿修改安装目录）
2. **重要**：关闭防火墙和杀毒软件
3. 安装完成后重启电脑
4. 首次运行需注册系统账号（与网站账号独立）
5. 输入系统序列号完成激活
6. 在系统设置中将IP地址修改为127.0.0.1

### 3.5 应用激活（以SecuMS为例）

1. 进入琏雾系统→点击"应用激活"
2. 选择SecuMS软件→点击下载
3. 输入应用序列号→等待5-7分钟下载完成
4. 点击"OS配置导入"→选择export.zip文件导入
5. 进入"应用部署"→点击部署按钮
6. 启动网关（切换为模拟状态→更新配置→开启网关）
7. 启动数据组件（更新→启动）
8. 完成应用授权

------

## 四、系统卸载

进入安装目录的IOT文件夹，运行Uninstall.exe按提示卸载。
