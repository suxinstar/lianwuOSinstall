# <div align="center">琏雾平台使用说明</div>

## **平台简介**

琏雾平台是由 **中国海洋大学智能物联网团队** 研发的智能物联网操作系统平台，支持在 **Windows 环境下** 快速开发和运行物联网应用系统，实现系统间互联互通，促进大规模物联网数据汇聚和人工智能发展。

**官方网站**： http://lianwu985.com

------

## **一、安装前注意事项**

### **1.1 运行环境冲突**

系统包含以下服务，安装前请关闭电脑中已存在的同类服务：

- MySQL
- Nginx
- RabbitMQ
- Redis
- EMQX

> 可通过任务管理器关闭相关进程，以防安装失败或端口冲突。
> 安装完成后可开启相关服务正常使用
------

## **二、使用流程概览**

1. 下载系统安装包（见3.1）
2. 访问琏雾平台网站（见3.2）
3. 注册平台账号（见3.2）
4. 申请系统序列号（见3.3）
5. 申请应用序列号（见3.3）
6. 安装并激活琏雾系统（见3.4）
7. 激活物联网应用（见3.2）

> **提示：** 系统默认使用期限为 **30 天**。

------

## **三、详细操作步骤**

> 更详细说明请参考琏雾平台官网：http://lianwu985.com

------

### **3.1 下载安装包**

1. 登录 GitHub 或访问官网 http://lianwu985.com
2. 下载 **LianwuOS-1.0.2.exe**（约 1.1 GB）

------

### **3.2 网站访问与账号注册**

1. 打开 http://lianwu985.com
2. 点击 **登录 → 立即注册**
3. 填写真实信息（姓名、电话、单位、邮箱）
4. 完成注册并登录平台

------

### **3.3 序列号申请**

#### （1）系统序列号申请

1. 登录平台 → **软件激活 → 序列号申请**
2. 选择数量并提交申请
3. 等待工作人员审核（1 个工作日内）
4. 在 **序列号管理** 中查看审核结果

#### （2）应用序列号申请（以 *SecuMS* 为例）

1. 进入 **应用商店** → 选择 *SecuMS* 软件
2. 点击 **申请序列号**
3. 等待审核通过（1 个工作日内）
4. 在 **序列号管理** 中查看序列号

------

### **3.4 系统安装与激活**

1. 双击安装包，按提示安装（**请勿修改安装目录**）
2. 关闭防火墙及杀毒软件
3. 安装完成后 **重启电脑**
4. 首次运行需注册系统账号（与网站账号不同）
5. 输入 **系统序列号** 完成激活
6. 在系统设置中将 **IP 地址** 修改为 `127.0.0.1`

------

### **3.5 应用激活示例（以 SecuMS 为例）**

1. 打开琏雾系统 → 点击 **应用激活**
2. 选择 *SecuMS* → 点击 **下载**
3. 输入 **应用序列号**，等待约 5–7 分钟完成下载
4. 点击 **OS 配置导入** → 选择 `export.zip` 文件导入
5. 进入 **应用部署** → 点击 **部署**
6. 启动网关（切换为模拟状态 → 更新配置 → 开启网关）
7. 启动数据组件（更新后启动）
8. 完成应用授权
9. 打开应用文件夹，运行启动脚本
10. 进入对应目录访问应用

------

## **四、系统卸载**

进入安装目录下的 **IOT 文件夹**，运行 `Uninstall.exe`，按提示完成卸载。
